    Introduction

    Welcome to News API documentation. You can use our API to access News API endpoints, which can get information on news, topics, etc in our database.

    Authentication

    News uses JWT tokens to allow access to the API. You can register a new user and use login API to get your JWT token.

    News expects for the API key to be included in all API requests to the server in a header that looks like the following:

    Authorization: Bearer yourtokenhere

    Or you can put your JWT token in query parameter like:

    http://localhost/api/news?token=yourtokenhere


    You must replace yourtokenhere with your personal JWT token.

    APIs

    POST api/login

    API login to get JWT Token.

    Example request:

    var settings = {
    "async": true,
    "crossDomain": true,
    "url": "http://localhost/api/login",
    "method": "POST",
    "headers": {
        "accept": "application/json"
    },
    "data":"email=name@email.domain&password=password"
}

$.ajax(settings).done(function (response) {
    console.log(response);
});

    HTTP Request

    POST api/login

    Example response:

    {
    "success":true,
    "code":200,
    "messages":[],
    "result":yourtokenhere

    HTTP Request

    GET api/news

    HEAD api/news

    Parameters

    Parameter Required Description
    token Required JWT token from login result.
    limit Optional Limit data per page. If not set, the default is 25.
    page Optional Page of collection. If not set, the default is 1.
    sort_by Optional Sort by field name. If not set, the default is id.
    Allowed fields: id, title, content, user_id, published_at, created_at, updated_at
    sort_type Optional Sort type of field name. If not set, the default is DESC.
    q Optional To search data by a keyword in fields title, content.
    status Optional To find specific status: deleted, draft, publish
    topic Optional To find specific by topic

    Errors

    The News API uses the following error codes:

    Error Code Meaning
    400 Bad Request – Your request sucks
    403 Forbidden – The data requested is hidden for administrators only
    404 Not Found – The specified data could not be found
    500 Internal Server Error – We had a problem with our server. Try again later.
    503 Service Unavailable – We’re temporarially offline for maintanance. Please try again later.
